# C.Odei Group Limited

A modern, professional website for **C.Odei Group Limited** — a premier construction and building materials company based in Accra, Ghana.

## About

C.Odei Group Limited has been a trusted name in the construction industry for over 20 years. We specialize in:

- **Construction Services** — Commercial and residential developments with professional work ethics
- **Building Materials Supply** — Premium iron rods, cement, blocks, nails, and binding wires

## Tech Stack

- **React 19** with TypeScript
- **Vite** for fast development and building
- **Framer Motion** for smooth animations
- **Lucide React** for icons
- **Tailwind CSS** for styling

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

The development server will start at `http://localhost:5173`
